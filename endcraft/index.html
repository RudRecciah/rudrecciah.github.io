What is EndCraft?

EndCraft is a Minecraft server I've owned with some friends for about # years. I've been working on and off developing and running the server over that time, which really doesn't yeild results. So, I decided to create this website; a place to leave public files and archives of EndCraft to help people with more drive than me develop other servers. Check out the PMMP Archive for MC:BE and Helios information and the Java Archive for open-source files for Spigot servers.

Past Development

EndCraft was originally developed for MC:PE in 2013 using PMMP. The server moved to being vanilla-based in 2017, coming back to PMMP in 2019. EndCraft also was ported to Java in 2019 using Paper. In early August 2020 I shut down Java for the last time as of writing and left the team maintaining MC:BE. A name change and few months later MC:BE EndCraft, known as Helios Practice, closed on Febuary 1st, 2021.
